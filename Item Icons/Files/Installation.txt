## Installation ##

Please note: This might not work with non-English Skyrim versions!

1. Place 'inventorylists.swf' in [Skyrim installation folder]/Data/Interface/inventory components/ (create if it doesn't exist)
2. Place 'craftingmenu.swf' in [Skyrim installation folder]/Data/Interface/ (create if it doesn't exist)

## Uninstallation ##

Simply remove the files!
